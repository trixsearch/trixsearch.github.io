## **🚀 Installation & Setup**  
Follow these steps to run the project locally:

```sh
git clone https://github.com/trixsearch/trixsearch.github.io.git
cd trixsearch.github.io
npm install
npm run dev
```

To build and deploy:  

```sh
npm run build
npm run deploy
```
